# Smart Agriculture System for Greenhouse Management

## Description
IoT-based smart agriculture system using Arduino UNO and ESP8266.
It monitors temperature, humidity, soil moisture, and water level,
and controls irrigation automatically using Blynk.

## Components
- Arduino UNO
- ESP8266 (ESP-01)
- DHT11 Sensor
- Soil Moisture Sensor
- Relay Module
- Water Pump

## Features
- Automatic irrigation
- Real-time monitoring
- Manual pump control via Blynk
